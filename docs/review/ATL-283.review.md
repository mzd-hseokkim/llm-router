### Warnings

- **`ActionButtons` 중첩 컴포넌트 정의** (`page.tsx:520`)
  `ABTestsPage` 함수 내부에 `ActionButtons` 함수 컴포넌트가 중첩 정의됨. 렌더 사이클마다 새 함수 참조가 생성되어 React가 매번 unmount/remount로 처리할 수 있음.
  설계 문서에서 "ActionButtons 인라인 또는 별도 컴포넌트" 양쪽을 허용했고, 외부 mutation 클로저(startMutation, pauseMutation 등)를 props drilling 없이 접근하기 위한 의도적 선택으로 보임. 이 규모에서는 실질적 성능 영향 없음.

- **다른 태스크 파일이 변경 목록에 포함** (`docs/review/ATL-290.review.md`)
  `git diff main..HEAD`에 ATL-290 리뷰 문서가 포함됨. 이 파일은 ATL-283 작업 범위 밖이므로 커밋 전 확인 필요.

### Info

- **타입 캐스팅 패턴** (`page.tsx:318, 457, 627, 632`)
  `as unknown as TrafficSplitWithModel[]` 패턴이 4곳에서 반복됨. `lib/api.ts`의 타입 불일치 때문에 발생하는 설계 의도적 패턴이며, 설계 문서 2.1절에서 명시적으로 언급됨. 향후 `lib/api.ts` 타입 개선 시 제거 가능.

- **구현 파일 미커밋 상태**
  `page.tsx`, `ABTestsPage.test.tsx` 등이 untracked 상태. PR 전 커밋 필요.

- **`routing/page.tsx`가 git diff에 표시됨**
