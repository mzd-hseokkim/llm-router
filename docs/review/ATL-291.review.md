### Info

- **`page.tsx:17`** — `state.replace("_", " ")` 는 첫 번째 언더스코어만 치환.
  현재 API 반환값(`closed`, `open`, `half_open`)에서는 문제없으나, 향후 확장 시 `replaceAll("_", " ")` 가 더 안전함.

- **`page.tsx:337`** — `cbResetMutation.isPending` 조건이 모든 카드의 Reset 버튼을 동시에 비활성화.
  단일 mutation 설계이므로 한 provider 리셋 중 다른 provider도 클릭 불가. 설계 문서에서 의도된 동작이나, UX 관점에서 사용자가 혼란을 느낄 수 있음.
