## Fixes Applied During Test Run

1. **`pages.test.tsx` — `vi.hoisted` for `mockConfig`**
   - `vi.mock` factory is hoisted above `const` declarations at compile time.
   - `mockConfig` was referenced inside the factory before being initialized (temporal dead zone).
   - Fix: wrapped with `vi.hoisted(() => ({...}))` so it's initialized before the factory runs.

2. **`pages.test.tsx` — `waitFor` for AlertsPage async queries**
   - `await act(async () => {})` flushes only one microtask tick, insufficient for React Query resolution.
   - Fix: replaced with `await waitFor(() => expect(...))` to poll until the query resolves.
