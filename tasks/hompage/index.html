Features

Everything your LLM stack needs

From startup to enterprise scale — the Gateway grows with you.

🧠

ML-Based Smart Routing

The Gateway learns from request patterns and automatically selects the optimal provider — balancing cost, quality, and latency for every call.

🔒

Zero-Exposure Key Security

Provider API keys are encrypted with AES-256-GCM and never leave the Gateway. Clients see only virtual keys — Provider credentials stay hidden.

💰

Cost Control & Chargeback

Per-team soft/hard budgets, real-time cost tracking, and automated chargeback reports. Know exactly where every dollar goes.

🏢

Enterprise Multi-Tenancy

4-tier Org → Team → User → Key hierarchy with RBAC, OAuth/SSO (Google, GitHub, Okta, OIDC), and isolated data per tenant.

⚡

Semantic & Exact Caching

Exact-match Redis caching and pgvector-powered semantic caching cut redundant LLM calls. Replay cached streams instantly.

🔌

MCP Gateway Hub

Central hub for Model Context Protocol — route tool calls, manage server registrations, and proxy WebSocket/SSE/stdio transports.

🛡️

Guardrails & PII Protection

Block prompt injections, mask PII before it reaches any model, and enforce content policies — at the gateway layer, before any token is spent.

🔄

Failover & Circuit Breakers

Automatic provider failover with weighted load balancing. Circuit breakers isolate failing providers so your service stays up.

📊

Full Observability

OpenTelemetry tracing, Prometheus metrics, immutable audit logs with GDPR/HIPAA retention policies, and Slack/email alerting.
